== Description ==

Set Sale price to a product or remove sale price from a product and it will be automatically added/removed to/from selected Sale Category.


== Installation ==

1. [Install and activate] Activate the plugin through the 'Plugins' menu in WordPress.
2. Create a Sale Category if you dont have one from Woocommerce Categories in order to select it as your sale category or select an existing one.
2. After installation go to "Set Sale Category" from dashboard menu and select your desired sale woocommerce category from the drop down list.
*Notes*
If you have already products on sale before the installation of the plugin, they will not auto assigned to selected product category. All you have to do is to re-update these products or manual set them into the desired sale category.

== Frequently Asked Questions ==

= I have already created a sale category (before installation of the plugin) and some products are assigned to it. Can i select the same category or i have to create new sale category?
You can select the already created sale category. 

= I have products on sale (before installation of the plugin) and they didn't auto assigned to selected category. What should i do? 
You have to re-update (just click the update button) these products in order to auto assigned or manually assign them to desired sale category.
